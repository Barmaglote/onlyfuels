1. Install node.js
2. Create project Server
	mkdir server
	cd server
	npm init -y		
3. Install express.js 
	npm install express --save
4. Install mongodb
5. Add mongodb to project
	npm i mongodb
6. Add packages
	npm i
7. Start server
	cd server 
	npm start
8. Start client
	cd client 
	npm start

Result in Browser:
	http://localhost:4200/
